Returns a random value following the distribution.

dstb

(dstb)(dstb {registry :registry, dstb-name :dstb-name, prng-param :prng, params :params, seed :seed, prng-name :prng-name, :or {seed 666, prng-name :xoroshiro128, dstb-name :uniform, registry distribution-registry}})

Returns a distribution created with the following parameters:

If provided, the following parameters are higher priority:

  • registry is where dstb-name will be searched for,
  • dstb-name is the name of the distribution as found in the registry,
  • prng the pseudo random number generator that is used to generate the distribution,
  • params the paramers of that distribution

If not provided:

  • seed is defaulted to 666,
  • prng is defaulted to a the prng named prng-name with seed,
  • dstb-name is defaulted to :uniform,
  • registry is defaulted with the distribution-registry.
